Sections are parts of the executable file where different types of data/code are stored.

Section	Meaning
.text	Contains the program’s executable code/instructions
.data	Contains initialized global/static variables
.rdata	Read-only data, such as strings and constants
.rsrc	Resources like icons, images, dialogs, version info

While imports are external functions the binary uses from DLLs.
